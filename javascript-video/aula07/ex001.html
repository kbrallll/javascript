Trabalhando com Operadores

Operadores Aritméticos

  • 5+2=7
  • 5-2=3
  • 5*2=10
  • 5/2=2.5
  • 5%2=1 ( Esse é o resto da divisão inteira )
  • 5**2=25 ( Esse símbolo é pontência )
  • Temos a seguinte expressão: 5 + 3 / 2 = 6.5 (Isso ocorre pq o terminal vai ler / e * primeiro)
  • Agora se for assim: (5 + 3 ) / 2 = 4 (Isso ocorre pq ele leu os () primeiro )

Ordem de Precedência no JS

  • ()
  • **
  • * / %
  • + -

Operadores Atribuição Simples

  • var a = 5 + 3 valor vai dar 8
  • var b = a % 5 valor vai dar 3
  • var c = 5*b **2 valor vai dar 45
  • var d = 10 - a/2 valor vai dar 6
  • var e = 6*2/d valor vai dar 2 (Nota que a ordem de precedência vem o que aparece primeiro)
  • var f = b%e+4/e valor vai dar 3

Auto-atribuições

  • var n = 3 (n = n+4) (valor que vai aparecer para n agora é 7)
  • n = n - 5 (Agora o valor de n é 2)
  • n = n * 4 (Agora o valor de n é 8)
  • n = n / 2 (Agora o valor de n é 4)
  • n = n**2 (Agora o valor de n é 16)
  • n = n % 5 (Agora o valor de n é 1)

Simplificando as Auto-atribuições do exemplo acima

  • n+=4 (ISSO SÓ DÁ PARA FAZER SE O N RECEBER O VALOR DE N MSM)
  • n-=5
  • n*=4
  • n/=2
  • n**=2
  • n%=5

Propriedade de Incremento

  • var x =5 (Valor de x é 5)
  • x = x + 1 (valor de x é 6) e poderia ser apresentado da forma simplificado (x+=1) Com o incremento fica (x++)
  • x = x - 1 (valor de x é 4) e poderia ser apresentado da forma simplificado (x-=1) Com o incremento fica (x--)
